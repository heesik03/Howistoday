## 오늘은 어때?

매일 확인해야 하는 날씨, 미세먼지, 환율 등의 필수 정보를 한눈에 확인할 수 있는 웹사이트입니다.
식사 메뉴, 영화 추천과 방명록 기능도 제공하고 있습니다.

## 🖥 웹사이트 페이지

- 메인페이지 (날씨, 미세먼지, 환율 정보)
- 소개
- 오늘뭐하지? (식사 메뉴, 영화 추천)
- 방명록
